# flyscroll

> 一款自定义滚动条插件

## Build Setup

1.使用方式

``` bash
# 安装依赖
npm install flyscroll --save

# 卸载依赖
# npm uninstall flyscroll

# main.js 使用方式
import FlyScroll from "flyscroll"
Vue.use(FlyScroll);

# *.vue 使用方式
<FlyScroll></FlyScroll>

```
